## обьекты класса 
> обьект, экзампляр, instance - обьект который создан по шаблону класса (он перенимает все атрибуты и методы класса)

```py
class A:
    var1 = 'переменная класса'

    def init(self):
        self.var2 = 'переменная обьекта'
```
